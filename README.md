# rd-132211
rd-132211 is the earliest release of Minecraft on the internet.

## What does rd mean?
"rd" stands for "RubyDung", which was another game created by Markus Persson (Notch) before Minecraft. RubyDung was used as a base for Minecraft's development.

## What does 132211 mean?
132211 is a `DDHHMM` timestamp representing the release date and time of this version (13th day, 22nd hour, 11th minute). In this context, "M" stands for "minute", not "month".
Because we know this version was built in May 2009, the full date and time is May 13, 2009, at 22:11 (10:11 PM) UTC+2.

## How to run
1. Ensure you have Java Development Kit (JDK) 8 or higher installed.
2. Ensure you have Apache Maven installed.
3. Clone this repository to your local machine.
4. Open a terminal and navigate to the project directory.
5. Build and run the project using Maven:
```
mvn clean compile exec:java
```
